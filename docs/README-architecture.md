# Архитектура приложения

## Обзор

Приложение построено на простой модульной архитектуре.

## Компоненты

### Основной модуль (main.py)

Является точкой входа в приложение. Отвечает за:
- Инициализацию приложения
- Вызов основной логики
- Обработку входных данных

### Конфигурация (pyproject.toml)

Содержит метаданные проекта, зависимости и настройки сборки.

## Паттерны проектирования

- **Простая структура** — минимум сложности
- **Модульность** — каждый компонент отвечает за свою задачу
