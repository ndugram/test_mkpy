# Руководство по началу работы

## Установка

Для установки проекта выполните:

```bash
pip install -e .
```

## Запуск

Запустите приложение:

```bash
python main.py
```

## Требования

- Python 3.13+
- pip
- [mkpy-client](https://github.com/ndugram/mkpy-client) — клиентская библиотека

## Структура проекта

- `main.py` — точка входа
- `pyproject.toml` — конфигурация проекта
